A secure backend authentication system built using Express.js and MongoDB, featuring OTP-based email verification, JWT authentication, and role-based access control (RBAC).

"Project Overview" :-
This project provides a complete authentication solution where users can:
Register with email verification (OTP)
Login securely using JWT tokens
Access protected routes based on roles (Admin/User)
Handle OTP expiration and resending

"Tech Stack" :-
Backend: Node.js, Express.js
Database: MongoDB, Mongoose
Authentication: JWT (JSON Web Token)
Email Service: Nodemailer (SMTP – Gmail/Mailtrap)
Security: bcrypt (password hashing)
